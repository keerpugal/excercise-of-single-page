SINGLE PAGE APPLICATION

Single Page Application(SPA) is an app(bundle of javascript code) which works with a browser.
SPAs load data/content from the server in background as per user demand.Think of the apps you use daily:
(Gmail, Google Maps & NetFlix). All these are examples of a SPA.
